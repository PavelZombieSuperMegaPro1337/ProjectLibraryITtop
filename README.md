# ProjectLibraryITtop
Практическая часть экзамена
=== БИБЛИОТЕЧНАЯ СИСТЕМА ==
компиляция:
g++ main.cpp -o biblioteka.exe
biblioteka.exe

что умеет программа:
1. Добавлять книги
2. выдавать книги
3. добавлять читателей
4. Принимать книги обратно
5. Сохранять в файл
6. Загружать из файла

правила:
- У читателя не больше 5 книг
- Книжку можно взять только если есть свободные
- ID начинаются с 1

важно:
- нажать "7" перед выходом чтобы сохранить
- Файлы knigi.dat и ludi.dat создаются автоматически

проблемы:
- Если не находит читателя - проверьте ID
- Максимум 100 книг и 50 читателей
